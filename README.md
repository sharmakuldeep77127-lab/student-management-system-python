# Student Management System

## Project Overview
This is a Python-based Student Management System that allows users to manage student records through a simple console interface.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Count Total Students
- Exit Program
- ## Technologies Used

- Python 3
- Lists
- Dictionaries
- Loops (while loop)
- Conditional Statements (if-elif-else)
- Functions
- Console-based User Interface

## Concepts Implemented

- CRUD Operations
  - Create Student
  - Read Student Data
  - Update Student Data
  - Delete Student Data

- Search Functionality
- Data Management using Lists
- User Input Handling
- Menu Driven Programming

## Future Improvements

- File Handling for Permanent Storage
- Student Grade Management
- Attendance Tracking System
- GUI Version using Tkinter
- Database Integration (MySQL)
